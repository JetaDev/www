Tarea 3

Crea los siguientes bloques:

HTML

Es el lenguaje de marcado que define la estructura de una página web. Utiliza etiquetas para organizar el contenido como texto, imágenes y enlaces.

CSS

Se encarga de la apariencia visual de una página web. Define colores, fuentes, tamaños y la disposición de los elementos. CSS hace que las páginas sean atractivas.

JavaScript

Es el lenguaje de programación que hace que las páginas web sean interactivas. Permite crear funciones dinámicas y responder a las acciones del usuario.

Backend

El backend es la lógica del servidor que gestiona datos y procesa solicitudes. Trabaja con bases de datos, autenticación y maneja la funcionalidad. Es donde ocurre la magia.
